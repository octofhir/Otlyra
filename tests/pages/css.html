CSS

This page is the CSS counterpart to defaults.html. Everything the <style> block above says is applied to the elements below, so what you see is what the cascade computed.

What works today

The cascade: the user-agent stylesheet, author rules from <style> and a style= attribute, resolved by origin, specificity and source order, with !important, inheritance, calc(), custom properties and media queries.

Selector matching, over our own DOM: type, class, id, attribute (including ^=, $=, *=), the four combinators, structural pseudo-classes, and :is(), :where() and :not(). The --dump-selectors flag answers "which elements does this selector match" against a real document.

What does not work yet

Layout reads twelve properties, so a rule the cascade computes correctly can still change nothing: borders, positioning, floats, flex and grid have no formatting context behind them yet. External <link> stylesheets are not fetched, and a restyle recomputes the whole document rather than the part that changed.
