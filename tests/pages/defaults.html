Default HTML rendering

Everything on this page is unstyled. What you see is the user-agent stylesheet, which is the browser's own opinion about what HTML looks like before any author has said anything.

Quotations and preformatted text

A block quotation is indented on both sides. It is one of the few elements whose default rendering people actually rely on.
fn main() {
    // In <pre>, whitespace   is   preserved
    // and lines    do not wrap however long they get.
    println!("hello");
}
