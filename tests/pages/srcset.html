Two densities

Red at one device pixel per CSS pixel, green at two.

A src beside a density list

Nothing in the list is for this density, so the plain address is what is left: blue, four hundred pixels wide and drawn at four hundred.

Widths against a stated size

Drawn a hundred and fifty across: the two-hundred file is the smallest that can fill it, so green, drawn at a hundred and fifty.

Widths against half the window

Nothing on offer is dense enough for half of a wide window, so the widest file wins and is drawn at the size it was asked for.

Widths with nothing said about the size

The whole window, which is what the specification says an absent sizes means — so the widest file, drawn the width of the page.

Alternatives in front of the picture

A wide window takes the first source; a format nothing can decode is passed over; the <img> is what is left when none of them apply.
